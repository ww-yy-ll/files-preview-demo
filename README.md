### 需要安装下面4个插件
### 预览 word 文件 docx
npm i docx-preview@0.1.4
npm i jszip
### 将页面html转换成图片
npm install --save html2canvas
### 将图片生成pdf
npm install jspdf --save

### ODF格式文件（7个插件依赖）
npm i js-sha1 --save
npm i jsrsasign --save
npm i js-md5 --save
npm i @lapo/asn1js --save
npm i sm-crypto --save
npm i ofd-xml-parser --save
npm i jszip --save
或者
npm install --save @lapo/asn1js js-sha1 jsrsasign jszip ofd-xml-parser sm-crypto js-md5
